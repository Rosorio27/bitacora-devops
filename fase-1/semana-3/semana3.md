## Rutas y jerquias del sistema de archivos de linux 

    La jerarquia standar (FHS - Filesystem Hierarchy Standard)

    Todo Linux sigue una convencion llamada FHS para que cualquier ingeniero, en cualquier distribucion, sepa donde encontra las cosas 

    /etc donde vive la configuraciones 

    /var donde el sistema respira (logs, errores, accesos, servicios)

        comando para ver la jerquia visual 

            tree -L 1
